The program is created for Lab#4(Utilizing stacks and queues in dealing with organizing a fiscal quarter worth of Spotify TOP 200 songs of the week.
The program is written in java and utilizes object classes for stacks and queues. 
The source code for the main, the stacks and queues, as well as the output file is uploaded with this README. 
Most of the program is commented or annotated. 
The program builds off of the last lab, where new functions and methods were added to incorporate the stacks and queues, as well as old functions replaced to accomodate the presence of multiple CSV files.
All of the above can be accessed with GitHub, and is written in java and compiled with the IDE IntelliJ.
