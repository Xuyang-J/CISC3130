The program is created for Lab#5(Utilizing binary search tree in dealing with organizing a Spotify TOP 200 songs of the week.
The program is written in java and utilizes object classes for nodes and BST.
The source code for the main, the BST and the node, as well as the output file is uploaded with this README. 
Most of the program is commented or annotated. 
The program builds off of the last lab, where new functions and methods were added to incorporate BST and node, as well as old functions replaced to accomodate the presence of CSV files.
All of the above can be accessed with GitHub, and is written in java and compiled with the IDE IntelliJ.
